# 🚀 SoftXet Task – Modern Frontend Landing Page

A sleek, animated, and performance-focused **Next.js 16 landing page** built with modern UI/UX practices.  
This project showcases smooth animations, reusable components, and a clean design system.

🔗 **Live Demo:** https://softxet-task.vercel.app

---

 ✨ Features

- ⚡ Built with Next.js 16 (App Router)
- 🎨 Beautiful UI using Tailwind CSS + shadcn/ui
- 🎬 Smooth animations using Framer Motion & GSAP
- 📱 Fully responsive (Mobile, Tablet, Desktop)
- 🧩 Modular component-based architecture
- 🚀 Optimized performance & fast load time
- 🔐 Type-safe development with TypeScript

---

 🧰 Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS, tailwind-merge, class-variance-authority
- **UI Components:** shadcn/ui, Radix UI
- **Animations:** Framer Motion, GSAP
- **Icons:** Lucide React
- **Fonts:** @fontsource/hind-siliguri
- **Package Manager:** pnpm
